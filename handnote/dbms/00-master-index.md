# DBMS — Master Index

> DBMS course-এর complete roadmap: beginner থেকে interview-ready পর্যন্ত।

---

## Course Overview

এই course-এ মোট **8টা chapter** থাকবে। Target:
- Fundamentals clear করা
- SQL hands-on করা
- GATE/Job/Interview সব কভার করা

---

## Chapters

| # | Chapter | What you'll learn |
|---|---|---|
| 01 | [DBMS Fundamentals & Architecture](01-dbms-fundamentals-architecture.md) | Data, database, DBMS, 3-schema, keys, constraints, ACID, basic SQL orientation |
| 02 | Relational Model, ER to Table Mapping | ER design, cardinality, participation, mapping rules |
| 03 | SQL Core (DDL, DML, DQL, TCL, DCL) | CREATE/INSERT/SELECT/UPDATE/DELETE, JOIN, GROUP BY, HAVING |
| 04 | Advanced SQL | Subquery, CTE, window function, ranking, set operations |
| 05 | Normalization & Functional Dependency | 1NF→BCNF, anomaly removal, decomposition |
| 06 | Transactions, Concurrency, Recovery | ACID, schedule, lock, deadlock, isolation levels |
| 07 | Indexing & Query Optimization | B+ tree, hash index, execution plan basics |
| 08 | NoSQL + CAP + Practical Design | key-value/doc/column/graph, partitioning, replication basics |
| 09 | [Interview Prep (Beginner to Advanced)](09-interview-beginner-to-advanced.md) | viva/interview Q&A ladder + practical SQL tasks |

---

## Study Flow

1. Chapter 01 দিয়ে foundation strong করো  
2. Chapter 02–04 দিয়ে modeling + SQL mastery  
3. Chapter 05–07 দিয়ে theory + internals  
4. Chapter 08 দিয়ে modern system view
