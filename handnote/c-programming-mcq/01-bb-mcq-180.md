**Q1.** What will be the output?

```c
#include <stdio.h>
int main() {
    int x = 5;
    printf("%d %d", x++, ++x);
    return 0;
}
```

- A) 5 7
- B) 6 6
- C) **Undefined behavior** ✅
- D) 5 6

> **ব্যাখ্যা:** Function arguments evaluate হয় unspecified order-এ। `x++` এবং `++x` দুটোই `x` modify করছে একই expression-এ, কোনো sequence point ছাড়াই → **undefined behavior**।

- C) 4 bytes
- D) Depends on compiler

### ⚠️ Exam-এ সবচেয়ে বেশি আসার Patterns

```
1. Undefined Behavior → x++ এবং ++x একই expression-এ
2. Post vs Pre increment → post আগে value দেয়, then modify
3. Bitwise operators → & AND, | OR, ^ XOR, ~ NOT, << >>, 
4. Operator precedence → () > ++ > * / % > + - > >> << > &
5. NULL pointer dereference → segmentation fault
6. Assignment in condition → if(x = 0) always false
7. sizeof() → char=1, double=8, int=4 (usually), char literal=4
```
