# Learning Handbooks

> **Author:** Md. Sahabuddin Hossain

Software Engineering topics নিয়ে সম্পূর্ণ **বাংলায়** লেখা handbook collection। Technical keywords ইংরেজিতে রাখা হয়েছে।

## Available Books

| # | Book | Topics | Target |
|---|------|--------|--------|
| 1 | **C Programming — Job Exam Guide** | 37 Topics + 100 Interview Q&A | BCS, Bank, IT Job |
| 2 | **Authentication & Authorization (.NET)** | OAuth, JWT, Identity | .NET Developers |
| 3 | **React for .NET Developers** | Components, Hooks, State | Backend → Frontend |
| 4 | **SSH Guide** | Keys, Config, Tunneling | DevOps, Server Admin |
| 5 | **SSL/TLS Guide** | Certificates, HTTPS | Security |

## কিভাবে পড়বেন?

- **বাম দিকের sidebar** থেকে topic select করুন
- **Search** (🔍) দিয়ে যেকোনো keyword খুঁজুন
- **Dark mode** toggle করতে উপরে theme icon ব্যবহার করুন
- **Print** করতে চাইলে browser এর print option ব্যবহার করুন

## Features

- 📖 বিস্তারিত ব্যাখ্যা + Diagram
- 💻 Code examples with syntax highlighting
- ❓ MCQ + Written problems with solutions
- ⚠️ Tricky parts & common mistakes
- 📝 Summary per topic
- 🔍 Full-text search
- 🌙 Dark mode support
